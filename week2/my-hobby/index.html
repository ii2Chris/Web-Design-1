My Hobby

It's Journaling!

How I got started

This past year, I have gotten into Journaling. It started as a way to keep track of my life, but became something I do every night.

The kind of Journal I use

Some reasons why I chose this journal to start with:
  • It feels premium
  • I like the design and dotted page format
  • It is highly customizable

In order to start journaling you don't need to spend money on a book like this, you can start however and whenever! This was a personal choice.
